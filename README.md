SPARK 2026 — Multi-Registration Finder

Static single-page app that lists students registered across Paper, Poster and Project categories.

Deployment

1. Commit and push this repo to GitHub.
2. Connect the GitHub repo in Vercel (or run `vercel --prod`).
3. Ensure Project Settings: Framework Preset = "Other / Static Site" and Root Directory points to the repo root.

This repo includes `vercel.json` to rewrite all routes to `index.html` for SPA routing.
